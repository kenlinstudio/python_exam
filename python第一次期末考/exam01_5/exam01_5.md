# 讀取檔案 + Set #

## 說明 ##

現有兩班的學生修數學與英文課
此兩班的修課資料存CSV檔中<br>
請讀取這兩個CSV檔<code>../app/math_list.csv</code>與<code>../app/english_list.csv</code> <br>
1. 請印出同時有修這兩門課的同學學號<br>
2. 請印出只有修一門課的學生學號<br>
3. 請印出有修A課沒修B課的學生學號<br>
4. 請印出有修B課沒修A課的學生學號<br>
5. 請列出所有學生的學號(不重覆)<br>

## Input Format ##

csv檔兩份

## Output Format ##

串列(list) (同時有修這兩門課的同學學號)
串列(list) (只有修一門課的學生學號)
串列(list) (有修A課沒修B課的學生學號)
串列(list) (有修B課沒修A課的學生學號)
串列(list) (所有學生的學號(不重覆))
