# 3023. 迴圈 + 判斷式 + 容器(list) I #

## 說明 ##

讓使用者輸入數字，直到-1結束<br>
請列出所有數字之後<br>
再列出由數字小到大排序的串列<br>
加總內容超過30的數字後輸出<br>

## Input Format ##

整數1<br>
整數2<br>
...<br>
...<br>
...<br>
-1<br>

## Output Format ##

串列(list)  (輸入的數字)<br>
串列(list)  (輸入的數字經過排列)<br>
整數(int)   (加總內容超過30<b>(>30)</b>的數字)<br>

## Sample Input 1 ##
```
50
40
30
20
10
-1
```

## Sample Output ##
```
[50, 40, 30, 20, 10]
[10, 20, 30, 40, 50]
90
```
