# 3069. 成對的括號字元 #

## 說明 ##

有些圖形化判面的程式語言編輯器可以自動標注程式碼中沒有成對的括號字元，如()、[]、{}、<>等<br>
方便程式設計師快速找到語法錯誤的地方，十分便利。<br>
在這個題目中，請寫一個程式來檢查輸入所給的一些字串其中的括號字元是否都成對。<br>
<br>
※ 106 台大資工系 大學部入學考題 02 題。<br>


## Input Format ##

第一行只有一個字N，代表接下來有幾組輸入。<br>
接下來的N行每一行都有一個字串，裡面可能出現大小寫字、數字、空白字元，還有()、[]、{}、<><br>
四種括號字元。每一行的字串長度最多為1024個字元<br>

## Output Format ##

總共有N行，依輸入順序依序輸出。<br>
每一行為驗證輸入中所給之字串的結果。<br>
若其中所有的括號字元均成對，則輸出一個Y字元。<br>
若至少有一個括號字元未成對，則輸出一個N字元。<br>

## Sample Input 1 ##
```
4
<[A()B]{}NTU>
())
()[1[2[3[[]
[CSIE}

```

## Sample Output 1 ##
```
Y
N
N
N
```
